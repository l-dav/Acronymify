How to use Acronymify?

You must first set up your local database. This database will contain all the terms that Acronymify will recognize.

In the config tab:

- you can link an online JSON file.

- you can enter your own custom words.

Once your database is set up, you can use Acronymify on any web page by selecting a word and using the shortcut to open its definition, or by clicking on the Acronymify popup.

Additionally, you can use the search bar in the home tab to explore your acronyms.
